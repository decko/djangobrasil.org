Como baixar o Django

Django é um software livre e está disponível sob a licença BSD. Ele necessita do Python em sua versão 2.6.5 ou superior, porém não há dependências de outras bibliotecas. Você pode baixá-lo de duas maneiras, veja:

Opção 1. Baixe a última versão oficial

A última versão oficial do Django é a 1.5.1. Para obtê-la, faça isso:

Primeiro baixe o arquivo Django-1.5.tar.gz. Então:

tar xzvf Django-1.5.1.tar.gz
cd Django-1.5.1
sudo python setup.py install

Você também pode utilizar o pip:

pip install Django==1.5.1

Opção 2. Baixe a última versão de desenvolvimento

A última e maior versão do Django está em um repositório Git (controle de versão). Baixe ela usando a linha de comando (você precisará ter o Git instalado):

git clone https://github.com/django/django.git

Ou, se preferir, você poderá baixar um arquivo zipado da última versão de desenvolvimento.

Depois de baixá-lo

Consulte o guia de instalação para mais instruções.
